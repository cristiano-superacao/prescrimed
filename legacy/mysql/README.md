# Legado MySQL

Este diretório reúne scripts e guias antigos usados durante fases anteriores do projeto em MySQL.

O fluxo atual suportado do PrescriMed está consolidado em PostgreSQL, tanto localmente quanto em produção no Railway.

Nada neste diretório é necessário para seed, testes de API, build do client, startup local ou deploy atual.

Mantenha estes arquivos apenas como referência histórica ou para consulta de migrações antigas.
